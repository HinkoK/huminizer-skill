# Huminizer Skill

Huminizer — это Agent Skill для глубокой редактуры русскоязычного AI-текста.

## Что делает
- Делает текст нативным и контекстно-уместным
- Убирает канцелярит, шаблонные вступления и «AI-стерильность»
- Сохраняет смысл, факты и функцию текста
- Учитывает жанр, канал, ритм и дистанцию до аудитории
